# Birthday Website

A delightful and interactive birthday website designed to celebrate special occasions with style and fun!

## Description

This project is a static HTML/CSS/JavaScript website that creates a personalized birthday greeting page. It features animated elements, a modal with an envelope animation, and customizable content to make someone's birthday extra special.

## Features

- **Interactive Design**: Click the button to open a surprise modal with an animated envelope
- **Visual Elements**: Includes birthday-themed images like cake, gifts, hats, and decorations
- **Responsive Layout**: Works on different screen sizes
- **Customizable Content**: Easy to personalize with different names, dates, and messages

## How to View

Simply open the `index.html` file in any modern web browser to view the birthday website.

## Customization

To personalize this birthday website:

1. **Change the Name**: Edit the `<h1>` tag in `index.html` that says "Dear Aluma" to the birthday person's name
2. **Update the Date**: Modify the date in the `.date-text` class (currently "18th of December")
3. **Customize the Message**: Replace the placeholder text in the letter inside the modal with your personal birthday message
4. **Replace Images**: Add your own images to the `_assets` folder and update the `src` attributes in `index.html`
5. **Styling**: Modify `style.css` to change colors, fonts, or layout as desired
